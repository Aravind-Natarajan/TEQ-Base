Desktop:
->Open Newfolder(Student Project)
->Open the Folder in VS Code
->Open Terminal

Create react project(Frontend):
->npm init vite
->Create project name (Enter file name) 'student'
->Select React
->Select JavaScript
->cd file name
->npm install
->npm run dev (run project command)

Create Node file(Backend):
->Create folder ('backend') in Student Project Folder
->Open the folder in VS Code
->Open Terminal

Create node-modules
->npm init -y
->npm install express mongoose cors axios nodemon body-parser
->create file in backend folder filename.js
->Open package.json
->"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start":"nodemon filename.js"
  }
->npm start (Run node server)
